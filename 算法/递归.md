## 递归
递归就是一个函数自己调用自己
函数内容就是循环体：相同的操作

函数参数是有规律的变化：循环中的每一步中值的变化
循环判断：到达某个条件则返回值，停止递归。

    1 * 2 * 3 * ... (n-1) * n 

    function fb(n) {
       if (n === 1) return 1 
       return n * fb(n -1) 
    }

    1 + 2 + 3 + 5 + 8 + 13 + 21 

    规律：
    a + b + c + d 
    a + b = c 
    b + c = d 
    前两个值相加得出弟三个数之和，所以要让循环下去，必须得有两个参数
    模版（相同）：n + m 

    function fb (n) {
        return a + fb()
    }
